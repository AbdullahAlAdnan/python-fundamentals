# Python Fundamentals

Python and SQL assignments covering core programming and data science 
concepts, completed as part of a Data Science program.

## Topics covered
- Module 2: Data structures — lists, dictionaries, sets, tuples, functions
- Module 3: OOP — classes, inheritance, encapsulation, polymorphism
- Module 4: SQL — SELECT, WHERE, GROUP BY, aggregations, LIKE, ORDER BY
- More modules added weekly

## Structure
Each folder contains clean, documented assignment solutions for one 
course module.
